#Command 1
 git --version
  Lets us know the current version of git installed on the system
#Command 2
 git clone
  Used to clone a Github repository to the local machine
#Command 3
 git add
  Used to add files to repository 
#Command 4
 git commit
  Used to commit changes made to a repository with a message
#Command 5
 git push
  Used to push changes made to the repository on the machine to the original repository in Github

git commands are given on the command line.You can clone a Github repository to your local machine.In Ubuntu,a directory with the same name as the cloned repository is made. You can add files to the repository using the terminal and commands and push the changes to the original repository on Github. You can also make changes to the file in Github and pull thechanges in the file to the local machine.
